_consistency_check(self)

source code 

Asserts some basic consistency of the Reaction instance.

  1. The number of substrates plus products equals the number of stoichiometric factors.
  2. With enough meta data (Compound formula, charge, or mass) stoichiometric balancing is checked.
Raises:
  • AssertionError - If the reaction is not well balanced.
