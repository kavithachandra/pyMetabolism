Parameters:
  • reactions (list) - A list of Reactions from which the graph can be populated.
  • name (str) - A simple string to identify the graph if desired.
  • weighted (bool) - By default the stoichiometric factors are used as edge weights, this can be turned off.
  • node_info (bool) - If true, information will be added to the graph's nodes. For compound nodes the following information will be tried in order:
    1. Compound.charge
    2. Compound.mass

    For reaction nodes Reaction.rate will be used.
