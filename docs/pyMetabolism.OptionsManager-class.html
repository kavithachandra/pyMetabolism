Class OptionsManager

source code

object --+
         |
        OptionsManager

This class unifies some global options, like certain name pre- and suffixes, number of cpus to use for certain parallel tasks, etc. This class is modelled as a singleton, i.e., only one instance of this class may exist.

To Do: Consider thread safety, though I would defer that to using this class.
