Module graph_algorithms

source code

Multiple ways of creating a graph representation of a Metabolism instance. Generation of a stoichiometric matrix is also possible.

Authors:
Nikolaus Sonnenschein, Moritz Beber

Copyright: Jacobs University Bremen. All rights reserved.

Since: 2010-04-04

Attention: Graph instances and Metabolism instances are not interlinked: Adding or removing reactions from one does not affect the other.
