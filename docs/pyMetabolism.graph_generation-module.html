Module graph_generation

source code

Multiple ways of creating a graph representation of a Metabolism instance. Generation of a stoichiometric matrix is also possible.

Authors:
Nikolaus Sonnenschein, Moritz Beber
Contacts:

Since: 2009-09-11

Attention: Graph instances and Metabolism instances are not interlinked: Adding or removing reactions from one does not affect the other.

random_metabolic_network(compounds, reactions, reversible, p, seed=None)

source code 

Creates a bipartite graph that models metabolism according to the principles of an Erdos-Renyi-like random graph.

Parameters:
  • compounds - Integer specifying the number of compounds.
  • reactions - Integer, specifying the number of reactions.
  • p - Probability for an edge to be set (on average this is the density of the graph).
Returns: BipartiteMetabolicNetwork
Raises:
  • ValueError - If either compounds or reactions are not integers.
